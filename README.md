# Student Course Enrollment System

## Project Overview

The Student Course Enrollment System is a MySQL database project designed to manage students, courses, enrollments, exams, and results. It demonstrates the use of SQL for database design, data insertion, and data retrieval.

## Technologies Used

* MySQL
* MySQL Workbench
* SQL

## Database Tables

1. Students
2. Courses
3. Enrollment
4. Exams
5. Results

## Features

* Create a student database
* Manage student records
* Manage course information
* Enroll students in courses
* Store exam details
* Store student results
* Retrieve data using SQL queries
* Generate grades using CASE statements
